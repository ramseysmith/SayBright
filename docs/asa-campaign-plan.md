# SayBright Apple Search Ads Campaign Plan

## Budget
$5 per day per campaign, matching the existing portfolio strategy.

## Campaign 1: Exact Match (Brand + Core Terms)
- Daily Budget: $5
- Match Type: Exact
- Keywords:
  - saybright
  - daily affirmations app
  - affirmations app
  - positive affirmations
  - morning affirmations
  - self love affirmations
  - daily positivity app
  - affirmation reminders
  - confidence affirmations
  - gratitude journal app

## Campaign 2: Broad Match (Discovery)
- Daily Budget: $5
- Match Type: Broad
- Keywords:
  - affirmations
  - positive thinking
  - daily motivation
  - self care app
  - mindset app
  - wellness daily
  - gratitude app
  - mental health daily
  - positivity
  - manifest app

## Negative Keywords (Both Campaigns)
- bible affirmations
- christian affirmations
- prayer app
- meditation music
- hypnosis
- subliminal

## Target CPA
- Initial target: $1.50 per install.
- Re evaluate after 100 installs using the conversion data in the ASA dashboard.

## Creative Sets
- Default: rely on App Store screenshots.
- Custom (if available): highlight the streak feature and the share cards.

## Launch Timing
- Start campaigns on day 1 of App Store availability.
- Run for at least 2 weeks before evaluating.
- Scale winning keywords, pause underperformers.

## Cross Promotion
- Add SayBright links in the settings screens of BlitzTap, CelebriDay, SignSnap, and Drift.
- Add "More Apps" links to the same portfolio in SayBright settings (already wired in Phase 5).
